Label a sentence/tweet as argumentative if it contains an Inference (what the author wants you to believe), regardless of truth or your agreement; if there is no Inference, it is No-Argument. Decide by asking: (1) what belief/claim is being pushed (Inference), (2) what facts/details are given (Information), and (3) whether the belief emerges from the facts; if Inference is present without explicit facts, label as Statement, and if Inference is supported by explicit Information in the text (e.g., dates, quotes/headlines, experiences, considerations, source references/URLs/quotations), label as Reason. Treat standpoints, rhetorical questions, some hashtags, and exception-phrasing (“not everyone…, but everyone…”) as strong Inference cues; treat concrete dates, quotes/headlines, and narrated experiences as Information cues; treat insults as not Inference (often None). For uncertainty, look up the tweet and thread context, but if still in absolute doubt, default to 0 (no component/no argument). Boundary examples: “Possible tangible benefit of #Brexit: a fairer immigration system.” = Statement (Inference-only assertion with implicit background, no explicit Information); a headline/link promo like “Mexico top court declares… unconstitutional… https://…” = Notification (Information-only summary, no Inference); “@sinnfeinireland Blah blah…” = None (neither Information nor Inference). Another boundary: an ad/donation tweet can still be Reason if it contains both a factual/consideration component and an Inference in a hashtag (e.g., #AbortionIsHealthcare) even if it initially reads as marketing.
