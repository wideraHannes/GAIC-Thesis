In TACO, a tweet is annotated as an Argument if it contains an inference, defined (via the Cambridge Dictionary) as a guess you make or an opinion you form based on the information you have. Argument tweets include Statements, where only inference is presented, and Reasons, where the inference is based on information included in the tweet (e.g., references), combining inference with information. Information is defined as facts or details about a person, company, product, etc., and is used to distinguish different kinds of argumentative tweets but does not by itself make a tweet argumentative. A tweet is No-Argument when it lacks inference, including Notifications that only provide information and None tweets that provide neither inference nor information.
